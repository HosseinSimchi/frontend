### Data Types

```javascript
let myVar: number;
let myVar: string;
let myVar: boolean;
let myVar: object;
let myVar: {
  name: string,
};
let myVar: string[];
let myVar: any[];
let myVar: [number, string];
```

```javascript

}
```

### Union Type

```javascript
const myVar: number | string;
```

### Literal Type

